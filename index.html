🕹️ Bem-vindo a Página de Jogos

Escolha um jogo dos quatro para se divertir!

🧠 Jogo Genius


O Genius é o clássico jogo eletrônico de memória. O objetivo é repetir sequências de luzes. A cada acerto, a sequência aumenta.

#️⃣ Jogo da velha


O Jogo da Velha é um clássico jogo de estratégia para dois jogadores. O objetivo é alinhar três símbolos iguais (geralmente "X" ou "O").

🎯 Jogo de Adivinhação de Número


Um jogo de adivinhação de números desafia os participantes a descobrirem um valor secreto pré-determinado. O jogador recebe dicas indicando se o número correto é maior ou menor.

✋ Jogo Pedra, Papel e Tesoura


O Pedra, Papel e Tesoura é um jogo rápido de escolhas. O objetivo é vencer o adversário escolhendo um símbolo que supere o dele em um ciclo onde todos têm vantagem e fraqueza.
